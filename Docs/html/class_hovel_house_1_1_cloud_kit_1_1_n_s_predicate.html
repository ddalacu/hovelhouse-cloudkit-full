Detailed Description

A predicate - A true/false evaluation that uses an apple-specific query syntax similar in purpose (but not syntax) to SQL

TBH the NSPredicate syntax is pretty bizzare. It's sorta half SQL and half Regex. I'd recommend reading a tutorial on it. https://nshipster.com/nspredicate/ Poorly formatter predicate strings will cause an exception to be thrown.
